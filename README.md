KissFFT-CS
==========

The aim of the project is to make KissFFT (http://sourceforge.net/projects/kissfft/) 
available in C# projects.

So far KissFFT-CS contains only the functionalities provided by the original methods 
kiss_fft and kiss_fftnd.

The KissFFT-CS is ported from C to C# 2.0. Visual Studio Community 2015 solution 
and project files are provided along with the source code.
